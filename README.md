У школьников, студентов, начинающих специалистов IT, экспертам по IT технологиям и IT компаниям 
с помощью веб-сайта “ITes”, который имеет: <br>
| Маркет-плейс | Новости МПИТ | Работать на заказ | Выполнение задач | Создание команды | Виртуальный кошелёк | Личный кабинет участника |

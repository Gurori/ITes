## 📰 Инструкция по установке React (вдруг пригодится)
React — это библиотека JavaScript с открытым кодом для создания внешних пользовательских интерфейсов. В этой инструкции будет написано, как установить React на вашеи компьютере.
### Шаг 1: Установка React
Необходимо ввести следующую команду в вашем терминале:
```
npm create vite@latest
```
Далее необходимо ввести название проекта, а потом выбрать React, затем JavaScript. После чего в командной строке нужно ввести поочерёдно следующие команды:
```
cd <название проекта>
```
```
npm install
```
```
npm run dev 
```
